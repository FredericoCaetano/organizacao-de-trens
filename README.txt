# Composição de Trens

Este é um programa simples para gerenciar a composição de trens, onde os vagões podem ser adicionados, removidos e visualizados.

## Funcionalidades

O programa oferece as seguintes funcionalidades:

1. **Adicionar Vagão**: Permite adicionar um novo vagão à composição de um trem específico.

2. **Remover Vagão**: Permite remover um vagão específico de um trem.

3. **Buscar Vagoes por Trem**: Encontra todos os vagões pertencentes a um trem específico.

4. **Contar Vagoes**: Conta a quantidade de vagões na composição.

## Como Usar

1. Execute o programa a partir do arquivo `main.py`.

2. Siga as instruções apresentadas no console para interagir com o programa.

3. Escolha uma das opções de funcionalidade e siga as instruções adicionais fornecidas.

## Requisitos

Este programa foi escrito em Python e não requer a instalação de bibliotecas adicionais.
